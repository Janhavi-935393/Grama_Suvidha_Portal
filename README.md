Grama Suvidha Portal
Grama Suvidha Portal is a modern Android application designed to bridge the gap between rural development projects and citizens. It serves as a digital gateway for rural infrastructure management, allowing administrators to update project progress while providing citizens with a transparent view of the work being done in their locality. The app supports multiple languages to ensure accessibility for all community members.

Features:
• Multilingual Support: Available in English and Kannada, making it accessible to local rural populations.
• Dual Access Modes: Includes an Admin Portal for officials to manage projects and a Citizen Portal for residents to track development and voice opinions.
• Project Tracking: Offers a detailed view of infrastructure projects, including status, location, and timelines.
• Feedback System: A dedicated module for citizens to submit feedback and suggestions on local projects.
• Offline Capabilities: Built with a local-first approach using Room database to ensure functionality even in areas with poor connectivity.
• Modern UI: Developed using Jetpack Compose and Material 3 design principles for a smooth user experience.

Technologies Used:
• Language: Kotlin
• UI Framework: Jetpack Compose
• Design System: Material 3
• Database: Room (for local data persistence)
• Architecture: MVVM (Model-View-ViewModel)
• Navigation: Jetpack Navigation Compose
• Local Preferences: Jetpack DataStore
• Image Loading: Coil

Setup Requirements:
• Android Studio: Iguana | 2023.2.1 or newer
• JDK: Version 7
• Android SDK: API Level 34 (Android 14) or higher
• Gradle: Version 8.0+

Installation Steps:
1. Clone the repository to your local machine.
2. Open the project in Android Studio.
3. Sync the project with Gradle files.
4. Run the app on an emulator or a physical Android device.
